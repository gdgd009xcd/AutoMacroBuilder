php windows server

1) download
   http://windows.php.net/download/
   x86 thread safe 
   
   解凍先　c:\php
   
2) http://www.microsoft.com/ja-jp/download/confirmation.aspx?id=48145
   visual studio c++ 2015 ランタイムダウンロード・インストール
   
3) 環境変数path にc:\php; 追加

4) php.ini-development を php.iniにコピー
　　extension_dir = "ext"
　　extension=php_mbstring.dll
　　extension=php_exif.dll
　　date.timezone = "Asia/Tokyo"

5) 起動・停止
　　(1) コマンドプロンプト起動
　　(2) cd c:\www (ドキュメントルートに移動)
　　(3) php -S localhost:8000
　　(4) Ctrl+C で停止。
　　
6) スクリプト
　　documentRootに下記内容の.batファイル配置。クリック起動。
:: server.bat
pushd %~dp0
php -S 192.168.11.35:8000
popd
